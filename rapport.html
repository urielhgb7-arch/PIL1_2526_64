3. Description des fonctionnalités

  • S'inscrire (/register) : création de compte utilisateur
  • Se connecter (/login) : authentification sécurisée
  • Profil : affichage et modification des infos personnelles
  • Compétences : ajout/suppression de compétences
  • Disponibilités : gestion du calendrier
  • Messagerie : communication entre membres

4. Technologies utilisées

  • Python / Flask
  • SQLAlchemy (base de données)
  • Render (déploiement)
  • GitHub (versioning)
